## Overview
Designed and developed a full-stack intelligent parking management system aimed at improving parking efficiency through data analysis and machine learning techniques. The system provides real-time parking information, management functions, and analytical insights for optimization.

## Tech Stack
Spring Boot · Vue.js · MySQL · Machine Learning · RESTful API

## Key Features
- Built backend services using **Spring Boot** with RESTful API architecture  
- Developed interactive frontend using **Vue.js** for real-time user interaction  
- Designed database schema using **MySQL** for parking data management  
- Applied machine learning methods for parking data analysis and optimization  
- Implemented modular system structure following MVC design principles  

## My Contributions
- Full-stack system design and implementation  
- Backend API development and database integration  
- Frontend interaction logic and UI implementation  
- System testing and performance optimization

## Outcome
The system demonstrates the integration of web engineering and data-driven analysis, providing a scalable solution for smart parking scenarios.
